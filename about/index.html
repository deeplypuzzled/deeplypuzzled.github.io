How this website was built

This website was built on a seriesof platforms provided by others. Here's a list of them, with the foundations at the bottom and the "higher" levels at the top.

  • Kramdown, the extended version of Markdown used for this website
  • Markdown, the simple coding for text to implement HTML formatting
  • Some HTML was required to create the basic structure...
  • ...and so was some CSS
  • (though a lot of the spadework was done by using a template, though I chose to use one that had been tested with GitHub Pages)
  • Ruby, the software that drives Jekyll
  • Jekyll, which converts plain text to HTML
  • GitHub Pages, the online documentation subset of GitHub
  • GitHub, the online/cloud version of Git (now owned by Microsoft)
  • Git, the leading version control system
